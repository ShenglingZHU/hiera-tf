# Hiera-TF

Multi-timeframe hierarchy signal framework with a static UI demo and Python/JS cores.

## Quick Start

**Static UI**:

```bash
python -m http.server 8000
# Open http://localhost:8000/apps/ui-static/index.html
```

**Python**:

```bash
pip install -e packages/htf-py
python -m htf.demos.demo_py_1
```

**JavaScript**:

```bash
node packages/htf-js/demos/demo-js-1.js
```
